# Commands
In the end the commands do work, the random art provides a random piece of art, the random color provides a random color, and the cat image command provides a random cat image, these I consider to be the main commands since the others are just to get the list of commands and such.
## Using the Commands
to use the commands you just do /legen in slack and the commands for legen come up.
